# Ejercicios: Facade Pattern

## ⭐ Nivel 1
Crear facade para sistema de cine (proyector, luces, sonido).

## ⭐⭐ Nivel 2
Facade para subsistema de notificaciones (email, SMS, push).

## ⭐⭐⭐ Nivel 3
Sistema bancario con facade para operaciones complejas (transferencias, préstamos).

## ⭐⭐⭐⭐ Nivel 4
Microservicios con facade que orquesta llamadas a 5+ servicios.
