# Ejercicios: SRP en Java y C#

## ⭐ Nivel 1
Implementa la misma clase `EmailValidator` en Java y C# usando características idiomáticas de cada lenguaje.

## ⭐⭐ Nivel 2
Refactoriza esta utility class en clases especializadas usando patterns de Java/C#:
```java
public class SystemUtils {
    public static void sendEmail(...) { }
    public static double calculateTax(...) { }
    public static void logError(...) { }
}
```

## ⭐⭐⭐ Nivel 3
Implementa sistema de gestión de productos usando Spring (Java) y ASP.NET Core (C#). Demuestra SRP con DI.

## ⭐⭐⭐⭐ Nivel 4
Compara implementaciones de mismo sistema en Java y C#. Analiza diferencias en adherencia a SRP según características del lenguaje.
