# Ejercicios: Arquitecturas Plugin

## ⭐ Nivel 1
Añade un plugin `CsvProcessor` al sistema de archivos. ¿Qué archivo necesitas modificar?

## ⭐⭐ Nivel 2
Sistema de validación de formularios con plugins (EmailValidator, PhoneValidator, DateValidator). Implementa descubrimiento manual.

## ⭐⭐⭐ Nivel 3
Sistema de autenticación con plugins (PasswordAuth, OAuthAuth, JwtAuth, BiometricAuth). Usa ServiceLoader para descubrimiento automático. Incluye configuración por plugin.

## ⭐⭐⭐⭐ Nivel 4
Framework de transformación de datos con plugins versionados, dependencias entre plugins, y validación de compatibilidad. Implementa hot-reload de plugins.
