# Ejercicios: Decorator Pattern

## ⭐ Nivel 1
Añade un decorator `CaramelDecorator` al sistema de café. ¿Qué clase necesitas modificar?

## ⭐⭐ Nivel 2
Sistema de pizza con decorators (Cheese, Pepperoni, Olives, Mushrooms). Implementa cálculo de precio y descripción.

## ⭐⭐⭐ Nivel 3
Sistema de logging con decorators (FileLogger, ConsoleLogger, DatabaseLogger, EmailLogger). Permite combinar múltiples destinos. Añade decorators de filtrado (ErrorOnlyDecorator, TimestampDecorator).

## ⭐⭐⭐⭐ Nivel 4
Framework de middleware HTTP usando Decorator. Implementa: Authentication, Compression, Caching, RateLimiting, Logging. Permite configurar pipeline dinámicamente.
