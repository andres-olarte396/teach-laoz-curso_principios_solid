# Ejercicios: OCP en TypeScript y Python

## ⭐ Nivel 1
Implementa `Shape` interface en TypeScript y ABC en Python. 3 formas geométricas con cálculo de área.

## ⭐⭐ Nivel 2
Sistema de validación extensible en ambos lenguajes. 4 validadores (email, password, phone, age) combinables.

## ⭐⭐⭐ Nivel 3
API REST de gestión de tareas:
- **TypeScript**: NestJS con DI
- **Python**: FastAPI con Depends
Demuestra extensibilidad añadiendo nueva funcionalidad sin modificar código existente.

## ⭐⭐⭐⭐ Nivel 4
Plugin system completo:
- **TypeScript**: Dynamic imports + decorators
- **Python**: Entry points + importlib
Implementa 3 plugins procesadores de datos. Soporte para hot-reload.
