# Ejercicios: OCP en Java

## ⭐ Nivel 1
Crea una interfaz funcional `Calculator` y 3 implementaciones (Add, Subtract, Multiply) usando lambdas.

## ⭐⭐ Nivel 2
Sistema de validación con cadena de validadores (composición funcional). Implementa emailValidator, passwordValidator, usernameValidator combinables.

## ⭐⭐⭐ Nivel 3
Framework de testing personalizado usando annotations y reflection (@Test, @BeforeEach, @AfterEach). Escanea clases y ejecuta automáticamente.

## ⭐⭐⭐⭐ Nivel 4
Plugin system con ServiceLoader + Spring Boot. Implementa 3 plugins de procesamiento de datos. Soporte para hot-reload y versionado.
