# Ejercicios: Event-Driven Design

## ⭐ Nivel 1
Añade un oyente `ShippingListener` al sistema de órdenes. ¿Qué clase necesitas modificar?

## ⭐⭐ Nivel 2
Sistema de gestión de usuarios con eventos (UserRegistered, UserLoggedIn, UserLoggedOut). Implementa 3 oyentes (WelcomeEmail, AuditLog, Analytics).

## ⭐⭐⭐ Nivel 3
Event bus con soporte para filtros y prioridades. Sistema de e-commerce con 5 tipos de eventos y 10 oyentes. Incluye eventos asíncronos.

## ⭐⭐⭐⭐ Nivel 4
Event sourcing para sistema bancario (AccountCreated, MoneyDeposited, MoneyWithdrawn, TransferCompleted). Reconstrucción de estado desde eventos, snapshots para performance.
